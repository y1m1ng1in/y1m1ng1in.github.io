Real-time
courier assistor

The real-time courier assistor is the software that helps couriers to find a optimized route to deliver packages.

The software allows couriers to scan packages and then an initial route to deliver those packages are generated. When the courier starts delivering, the software detects road interrupts based on the real-time road conditions and adjust route.

The software also provides courier with summarys of delivering when all the packages are delivered or delivering ended up with some packages are not delivered. The route of the delivery, the packages information and other related messages are recorded.

Real-time courier assistor is the software which is implemented as the project of object-oriented programming course (2018 Spring) in Portland State University.

This software is one of the most impressive projects to me since I began to study computer science. It takes three weeks to design, implement and test, it requires a lot of knowledge of graph theory. It is also the first project of mine to solve real world problems.

To provide couriers with convenience, the software allows couriers to check current packages on the vehicle at any time. Couriers can also check current route and predetermined route at any time.

When an interrupt occurs such as a road closure, the software prompt couriers that another quicker route is detected which can save couriers' time.

The software provides couriers with delivery interface that can be used when a certain location arrive, the package is unloaded by that interface.

Finally, the delivery summary is showed on the left-top of the picture. The overall route is recorded and all the traffic conditions during the delivery process has been recorded.
